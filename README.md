## 使い方

### ゲームを追加する
- タイトルを入力
- ゲームの実行ファイルを参照
- セーブフォルダ又はセーブファイルを参照

![](docs/images/game.png)

ErogameScape -エロゲー批評空間- のデータベースから登録も可能

![](docs/images/database.png)

### ゲームを起動する
起動時にローカルと同期フォルダのセーブデータの最終更新日を比較し、最新のセーブデータをダウンロードします

### プレイ履歴を見る
ゲーム画面がアクティブ時のみ時間をカウントし、同期時にプレイ時間を記録します。

![](docs/images/gamelog.png)
